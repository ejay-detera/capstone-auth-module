## System Overview

The project is a decoupled authentication system consisting of the following components:
- **Backend**: Laravel 13 (PHP 8.3)
- **Frontend**: Vue 3 (Vite + TypeScript)
- **Persistence Layer**: MySQL 8.0
- **Caching Layer**: Redis
